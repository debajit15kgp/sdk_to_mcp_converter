## Configuration

Set the following environment variables:

- `GITHUB_TOKEN`: github token
- `GITHUB_ACCESS_TOKEN`: github access token

Or create a `config.json` file:
```json
{
    "credentials": {
        "api_key": "your_api_key_here"
    }
}
```
